## Midjourney 在国内使用，有一个天然的门槛

Midjourney 不像 ChatGPT 有自己的网站和 App。它是**跑在 Discord 上的**。

Discord 是什么？一个国外的聊天软件，在国内需要翻墙才能用。

所以你想用正版 Midjourney，需要：

1. 科学上网
2. 注册 Discord 账号
3. 加入 Midjourney 服务器
4. 订阅付费套餐（$10-60/月）
5. 在 Discord 聊天框里输入 `/imagine` 指令生图

对很多人来说，光"科学上网 + Discord"这两步就够折腾的了。

## 官方版 vs 国内镜像版

| | 官方版 | 国内镜像版 |
|:---|:---|:---|
| 需要梯子 | ✅ 需要 | ❌ 不需要 |
| 需要 Discord | ✅ 需要 | ❌ 不需要 |
| 操作方式 | 聊天框输指令 | 网页直接操作 |
| 模型版本 | V6 | V6 |
| 价格 | $10-60/月 | ¥29/月起 |

镜像版用的是 Midjourney 的 API 或者共享账号，效果和官方版**完全一样**，只是操作界面不同。

## 适合什么人

- **不想折腾翻墙和 Discord 的**：直接镜像版，打开网页就能画
- **偶尔画着玩的**：镜像版月付 ¥29，比官方 $10 便宜
- **专业设计师/重度用户**：建议官方独立账号（标准版 $30/月），镜像版是共享账号，使用频率受限

## 快速上手指南

不管你选哪个版本，Midjourney 生图的核心都是一个东西：**提示词（Prompt）**。

新手可以这样开始：

1. 先想到你要画的画面
2. 用简单的英文描述（不会英文用翻译软件）
3. 加上风格词：`--style raw`（写实）、`--ar 16:9`（横版比例）

比如想画一只猫：
```
a cute orange cat sitting on a windowsill, sunlight, cozy atmosphere --ar 4:3
```

多试几次就找到感觉了。AI 绘画没有门槛，唯一需要的是一点点耐心。
