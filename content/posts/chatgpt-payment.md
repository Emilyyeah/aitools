## 先确认：你真的需要 Plus/Pro 吗

**Plus（$20/月，约¥140）**：GPT-5 + GPT Image 2 + 更高频率上限 + 优先体验新功能
**Pro（$200/月，约¥1400）**：无限使用 + 最高优先级 + 研究级功能

如果你只是每天聊几句、查资料、偶尔写东西，免费版（或聚合平台免费额度）就够了。别为「开了会员就变强」的幻觉付费。

## 方案一：代充（最快）

**原理**：你把支付页面的授权信息给代充平台 → 平台用海外正规卡帮你付款 → 你支付宝/微信付人民币。全程**不需要给密码**。

### 推荐平台

#### DGT Store

- 直达：[dgtsell.com](https://dgtsell.com)
- Plus：¥140-145/月 · Pro：¥1350-1450/月
- 优势：购买流程像电商一样简单，账单截图可查，退款政策明确
- 适合：追求标准化流程的用户

#### 银河录像局

- 直达：[yinhe.tv](https://yinhe.tv)
- Plus：¥140-150/月
- 优势：2019年成立，工信部 ICP 备案，国内最老牌，售后最稳
- 适合：追求稳定、看重平台资质的用户

### 操作步骤（以 DGT Store 为例）

1. 打开 [chatgpt.com](https://chatgpt.com)，登录你的账号
2. 左下角 Settings → **Upgrade to Plus**
3. 到支付页面后，**复制页面 URL**
4. 打开 [dgtsell.com](https://dgtsell.com)，找到「ChatGPT Plus 代充」
5. 粘贴 URL → 选择套餐 → 支付宝/微信付款
6. 等 2-5 分钟，平台完成代充，你的账号变 Plus
7. 平台提供账单截图确认

**如果页面要你输入信用卡信息**：说明你还没到正确的支付页面。代充需要的 URL 是已经生成了支付会话的页面，不应该是让你填卡号的页面。如果卡在这一步，换一个网络环境或浏览器重试。

## 方案二：Apple 内购（自己操作，仅限 iPhone/iPad）

不信任代充平台？自己动手。通过美区 Apple ID + 礼品卡余额在 ChatGPT iOS App 里订阅。

### 第一步：注册美区 Apple ID

1. iPhone **设置 → 通用 → 语言与地区**，改成**美国**
2. 打开 App Store，退出当前账号
3. 找一个**免费 App**（如 ChatGPT），点「获取」
4. 弹出登录界面 → 点「创建新 Apple ID」
5. 填写信息：英文姓名、**全新邮箱**（Gmail 或 Outlook）、密码
6. 手机号：**可以用 +86 中国号码**
7. **付款方式选「None」**（如果没出现这个选项，说明你在付费 App 页触发的注册，返回找免费 App 重来）
8. 填写美国地址：搜「美国地址生成器」，选一个**俄勒冈州（Oregon）**的地址——俄勒冈免消费税

**常见踩坑**：
- 没有「None」选项 → 必须在免费 App 页面触发注册
- 邮箱已注册过 Apple ID → 用全新的
- 「无法创建账户」→ 关掉 VPN 用 Wi-Fi 或 4G 重试

### 第二步：买礼品卡 + 充值

1. 打开**支付宝** → 搜「出境」→ 折扣礼卡 → iTunes / App Store
2. 选面额 **$25 或 $30**（$20 刚好一个月不够，留余量）
3. 付款后会收到兑换码
4. 打开 App Store（确认登录的是美区 ID）→ 点头像 → **兑换礼品卡或代码** → 输入兑换码
5. 余额到账（在账户页能看到）

⚠️ **别在淘宝/闲鱼买超低价礼品卡**，大概率是黑卡，用了美区 ID 会被封。

### 第三步：订阅

1. 在美区 App Store 下载 ChatGPT App
2. 登录你的 ChatGPT 账号
3. Settings → **Subscription → Upgrade to Plus**
4. 确认订阅，从 Apple ID 余额扣款
5. 实际扣 $22-23（官方 $20 + 苹果汇率差/税费）

### 长期维护

- 每月自动续费，余额不够会暂停
- 余额快用完时，再买礼品卡充值
- 扣款时不需要登录美区 ID（订阅自动关联），但余额不足时会发邮件提醒
