## 场景化选型

### 日常问答 + 学习

**ChatGPT**。中文最自然，回答最完整，免费版也够用。

### 写代码 + Debug

**Claude**。代码生成和审查能力接近人类高级工程师。配合 Claude Code 效果翻倍。

### 免费体验 + 长文档

**Gemini**。100 万 token 上下文 = 一次塞进 3 本书。任何需要处理超长文本的场景首选。而且免费额度最大方。

### 内容创作（公众号、文案）

**ChatGPT**。中文语感最好，会写「人话」。创意脑暴也更强。

### 分析视频/音频

**Gemini**。原生多模态，直接上传视频文件分析内容、生成字幕、提取要点。

### 不确定，都想试试

**聚合平台**。注册一个账号，三个模型随便切换。不用纠结选哪个——都试试，哪个顺手用哪个。
