## API 中转

主流中转站都已支持 Claude 全系列（Opus 4.5、Sonnet、Haiku）。代码里改 `base_url` 即可。

## Claude Code

Anthropic 2026年推出的命令行 AI 编程助手。

安装后，通过中转平台获取 API Key，配置后在终端直接跟 Claude 对话。写代码、调试、重构全在命令行里完成。

## 适合谁

- 需要 API 集成的开发者
- 想在终端用 AI 辅助编程的
- 团队需要统一模型接入的

## 怎么选中转平台

- 看 SLA 保障（99.9%+）
- 看是否支持 Claude 最新模型
- 看有没有免费额度测试
- 看代码示例是否齐全
