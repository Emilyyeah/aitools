## 为什么选这条路

- 不把账号交给第三方代充平台
- 不走已经萎缩的虚拟卡（WildCard 已停服）
- 苹果官方结算通道，封号风险最低
- 缺点：只限 iPhone/iPad 用户

---

## 完整操作步骤

### 第一步：注册美区 Apple ID（5分钟）

> ⚠️ 必须用**全新的邮箱**（没注册过任何 Apple ID 的），推荐 Gmail 或 Outlook。

**手机端注册（推荐，成功率高）**：

1. 打开 **设置 → 通用 → 语言与地区**，把地区改成**美国**（注册完可以改回来）
2. 打开 App Store，点右上角头像，滑到底部点**退出登录**
3. 随便找一个免费 App（比如 ChatGPT），点**获取 → 安装**
4. 弹出登录界面时，点**创建新 Apple ID**
5. 填写信息：
   - 姓名：英文拼音（如 Zhang San）
   - 邮箱：全新的、没用过的
   - 密码：至少 8 位，含大小写字母和数字
   - 手机号：**可以用 +86 中国大陆手机号**
6. 验证邮箱和手机验证码
7. 关键步骤——**付款方式选「None」（无）**
   - 如果没出现 None 选项：说明你是在付费 App 页面触发的注册，返回找一个**免费 App**重新触发
8. 填写美国地址：
   - 搜「美国地址生成器」找一个免税州的真实地址（推荐**俄勒冈州 Oregon**、**特拉华州 Delaware**、**阿拉斯加州 Alaska**，这几个州免消费税）
   - 城市、街道、邮编、电话（用生成的即可）全部填上

**常见踩坑**：
- 「无法创建账户」→ 网络问题，关掉 VPN 或用 Wi-Fi 重试
- 没有「None」选项 → 在免费 App 页面触发注册（不是在付费 App 页面）
- 邮箱已被注册 → 用全新的邮箱

### 第二步：买 App Store 礼品卡

**支付宝路径**（最方便）：

1. 打开支付宝 → 左上角切换到**美国**地区（或搜「出境」进境外服务）
2. 找到**折扣礼卡 → iTunes / App Store**
3. 选择面额（建议买 **$25 或 $30**，$20 刚好一个月不够，留点余量）
4. 付款后会收到一串兑换码（通常在支付宝订单详情或短信里）
5. **⚠️ 只买官方渠道（支付宝出境、苹果官网）的礼品卡，淘宝/闲鱼的超低价礼品卡大概率是黑卡，兑换后美区 ID 会被封**

**其他渠道**：
- Amazon.com 买电子礼品卡（需要美亚账号 + 国际信用卡）
- 苹果官网买礼品卡（需要国际信用卡）

### 第三步：兑换礼品卡

1. 确保 iPhone 登录的是**美区 Apple ID**（看 App Store 右上角头像确认）
2. 打开 App Store → 点右上角头像 → **兑换礼品卡或代码**
3. 手动输入或相机扫描兑换码
4. 兑换成功后会显示余额（在账户页面顶部能看到）

### 第四步：订阅 ChatGPT Plus

1. 打开 ChatGPT App（在美区 App Store 下载）
2. 登录你的 ChatGPT 账号
3. 进入 **Settings → Subscription → Upgrade to Plus**
4. 选择订阅计划，确认付款
5. 系统从 Apple ID 余额扣款，不需要信用卡

**首次订阅扣款金额**：官方 $20/月 + 苹果可能多扣约 $2-3（税费/汇率差），实际约 $22-23/月。

## 长期维护

- **续费**：每月自动从余额扣，余额不够会暂停订阅。提前充够钱。
- **充值**：金额快用完时，再买礼品卡充值进去。
- **别频繁切换 Apple ID**：美区 ID 下载完 App 后可以切换回国区 ID 日常使用，但订阅扣款时需要美区 ID 登录。如果扣款失败（连续三次），订阅会被取消。
