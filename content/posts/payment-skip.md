## 这个方案适合谁

- 不想搞海外支付（信用卡、礼品卡、代充统统不想碰）
- 不想花 $20/月（¥140）开 Plus
- 不介意通过第三方平台使用（不是官方直连）
- 想一个平台同时用 ChatGPT + Claude + Gemini

---

## 原理

聚合平台在国内部署服务器，自己买了 OpenAI/Anthropic/Google 的 API 额度，包装成网页版 ChatGPT 给你用。

你付的钱是「在平台使用 AI 的额度」，不是「给 OpenAI 的订阅费」。对 OpenAI 来说，你根本不存在（平台是 OpenAI 的客户，你是平台的客户）。

**好处**：支付问题完全消失。支付宝/微信直接付，¥30 起步。
**代价**：平台在中间能看到你的对话内容。不是官方直连。

## 怎么操作

1. 搜索「ChatGPT 聚合」「AI 聚合平台」「ChatGPT 中文版」
2. 注册几个平台，比较一下：
   - 支持哪些模型（GPT-5？GPT-4o？Claude？Gemini？）
   - 免费额度多少
   - 界面好不好用
   - 付费套餐价格
3. 先白嫖几天免费额度，满意了再付费
4. 付费选最便宜的套餐就行（不用一上来买年卡）

## 怎么选聚合平台

### 硬指标

- 网站底部有 **ICP 备案号**（正规企业）
- 支持**多个模型**（ChatGPT + Claude + Gemini 都在）
- 有**免费试用额度**（不花一分钱先试）
- 回复速度和官方 ChatGPT 差不多（如果明显慢，说明平台的 API 额度紧张）

### 加分项

- 支持**独立会话**（你的对话和其他用户隔离）
- 支持**文件上传**（PDF、图片等）
- 有**手机 App**（比网页版方便）
- 有用户社群可以看口碑

### 红线

- 运营不到半年的新平台 → 等半年再看
- 搜索「平台名 + 跑路」有大量负面 → 别碰
- 要求充值大额才能用 → 先用小额试
